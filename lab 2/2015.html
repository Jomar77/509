Analysis of the website:2015

2015 design had some changes compared to 2010. One major is showcasing the badge of the day which highlights an achievement from a particular game. we can also see hot new games and highest rated games as a feature in this design. In terms of technicality, the banner in which they showcase the badges seems to wrapped in a container which also contains a list. This enables the banner to be scrollable.

Upon scrolling down the website, we can see the updated design for the featured games. On the right side of the website, we can see that there is an additional highlights which would be the top games of this month. Underneath it would be the notable games section which showcases the trending games that deserves recognition.
